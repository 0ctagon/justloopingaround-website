Marc Rebillet Live Stream Repertoire

Répertoire de chansons créées par Marc Rebillet lors des live streams.

MENU

Ce site répertorie toutes les chansons que Marc a fait dans ses live streams depuis qu'il a commencé à streamer sur YouTube.
Il y a (presque) tous les streams de Youtube et Twitch, totalisant plus d'un millier de chansons.

Une Tierlist de toutes les chansons avec mon avis biaisé est disponible sur la page Tierlist.

Vous pouvez également trouver une page de statistiques détaillées, analysant le système de classement, mes biais dans la notation et certaines des propriétés des chansons.

Pour mieux se retrouver la multitude de chansons disponibles, le site propose:

  • une Stream Highlights section listant quelques chansons célèbres/agréables pour commencer,
  • un Jukebox qui donne une chanson aléatoire, extremement pratique pour en découvrir,
  • un Chansons Explorer qui permet de trouver n'importe quelle chanson grace à une barre de recherche,
  • la liste de tous les lives rangée par ordre de parution.

Bonnes découvertes !

Jukebox

Ce jukebox vous propose des chansons aléatoires parmis toutes les chansons que Marc a stream depuis ses début sur Youtube. Vous pouvez parametrer les chansons que le jukebox propose en cliquant sur l'icone de parametre. Une fois la chanson choisie, vous pouvez aussi cliquer sur le nom du live pour afficher les autres chansons du live. Il y a actuellement [CHARGEMENT] differentes chansons possibles dans le Jukebox !

Cet outil vous permet de rechercher des chansons à l'aide de mots-clés. C'est l'outil parfait pour rechercher une chanson coincée dans votre tête, ou celle dont vous ne connaissez que le titre. Vous voulez savoir combien de chansons parlent de sexe ou combien ont été étiquetées « funk » ? Cet outil est également très utile pour cela (64 et 107 respectivement).
